Janus has support for multiple compiler and MPI toolchains, all of which are provided using the module command.

Compilers

There are three compiler suites on Janus. Note: if you swap compiler suites you should first unload the previous compiler module before loading a new module.

We highly recommend using the Intel Compiler Suite when compiling applications to be run on Research Computing resources.

$ module load intel/intel-13.0.0

MPI

There are four MPI implementations on Janus. We recommend Intel MPI.

$ module load intel/impi-13.0.0

All MPI implementations available in the Research Computing Environment provide standard wrapper commands.

Language Wrapper command
C mpicc
Fortran mpifc, mpif77, mpif90
C++ mpicxx

The module you load is dependent on the compiler suite you have loaded. For example if you have the Intel compiler loaded you should load the MPI implementation built with that compiler.

Janus compile nodes

Dedicated nodes are provided for compiling software meant to run on the Janus compute cluster. Access to these nodes is provided from any login node via SSH.

$ ssh janus-compile1

The Janus compile nodes are otherwise identical to Janus compute nodes and are most likely to produce problem-free builds, especially for applications that use MPI or are heavily hardware-optimized.
