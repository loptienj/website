Quick Start

This is a quick-start guide to getting up and running in the Research Computing environment. This guide describes how to request an account, the process to obtain an authenticator device needed to log in, how to log into the RC environment and some important details about the storage and modular software environment.

Users of Research Computing resources are responsible for complying with all university policies, including (but not limited to) those defined at http://oit.colorado.edu/policies/cu-boulder-it-policies.

Get an account

Access to Research Computing resources is available to CU-Boulder, CU-Denver, and NCAR users. If you are a CU user, request access using the RC Account Request portal. If you are an NCAR user, request access through the CISL allocation request system.

External users must obtain a sponsored affiliate account with CU before they will be able to request a Research Computing account.

Get and register an authenticator

Access to Research Computing resources requires two-factor authentication. If you are an NCAR user, use your Yubikey or Cryptocard to authenticate. If you are a CU user, request a physical one-time-password (OTP) generator device by either

  • filling out the OIT Authenticator order form, or
  • emailing your request to rc-help@colorado.edu.

Be prepared to show your BuffOne card or other state-issued ID when you pick up your authenticator.

Once you’ve received your OTP authenticator, register it at the OTP Self Service portal, and verify that it works by following the instructions on the OTP Login Test page.

Log in

Access to Research Computing resources is primarily provided using the secure shell (SSH) protocol. The ssh command is available on virtually any Linux or Mac OS X system. If you are a Windows user, we recommend that you use the PuTTY ssh client application, though alternatives exist.

Request an allocation

Access to Research Computing computational resources is moderated by finite allocations of compute time. If you are working with a PI who already has an active allocation, have them email us at rc-help@colorado.edu to request that you be added to it. Otherwise, use the Allocation Request page to request a ‘startup’ allocation, available to each new user. Otherwise click below to request your own allocation.
