Affiliates of NCAR have access to the Janus compute cluster through CISL.

Apply for an allocation

Submit a request for an allocation on the Janus cluster using the CISL allocation request process. Once your request is approved, you will receive login details, including your RC username.

When you log into RC systems, authenticate to your RC account using your NCAR Yubikey or CryptoCard. (More information on logging into RC systems is available in the user guide, and later in the getting started guide.

Next steps

Now that you have an RC account, you're ready to get started with the specific RC resources that you're interested in using. You should also read the CISL "Quick start for NCAR users."

I want to run compute applications on the shared infrastructure.
The Janus HPC cluster, Crestone bladecenter, GP-GPU nodes, large-memory nodes, and other compute resources are available to the RC community.
I want to use dedicated nodes in the Research Computing condo.
Dedicated nodes may be purchased as part of the Blanca condo cluster.
I want to store or access data in the PetaLibrary.
The PetaLibrary Active and PetaLibrary Archive storage services can be used to store data for compute or long-term storage.
