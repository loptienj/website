MPI

There are four MPI implementations on Janus. Once again we recommend the Intel MPI implementation. No matter which implementation you choose, they all provide MPI helper programs:

Language Compiler
C mpicc
Fortran mpifc, mpif90, mpif77
C++ mpicxx

The module you load is dependent on the compiler suite you have loaded. For example if you have the Intel compiler loaded you should load the MPI implementation built with that compiler.

Intel MPI (IMPI)

You need to load the IMPI module, the latest version is 15.0.1, which corresponds to the latest Intel compiler suite.

$ module load intel/impi-13.0.0

OpenMPI

Once again you need to load the OpenMPI module, the latest is version 1.8.4. If you are using the Intel suite:

$ module load intel/intel-15.0.1 $ module load openmpi/openmpi-1.8.4_intel-13.0.0

Or if you happen to use the GNU compiler suite:

$ module load gcc/gcc-4.9.2 $ module load openmpi/openmpi-1.8.4_gcc-4.9.2

MVAPICH2

The latest version is 1.9. This implementation has only been built with the Intel and GNU compiler suites.

$ module load mvapich2/mvapich2-1.9_intel-15.0.1

MPICH

The latest version is 3.1.2. This implementation has only been built with the Intel compiler suite.

$ module load mpich/mpich-3.1.2_intel-13.0.0
