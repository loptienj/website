Software Compilers

There are three compiler suites on Janus, their use is highlighted below. They are all obtained by loading the appropriate module. Please note, if you swap compiler suites you should unload the previous compiler before loading a new one. This can be done with the module unload command.

We highly recommend using the Intel compiler and MPI suites.

Intel Compiler Suite

You should load the latest intel module, this is version 15.0.1.

$ module load intel/intel-15.0.1

This will provide you with the following compilers in your path

Language Compiler
C icc
Fortran ifort
C++ icpc

The Portland Group

Once again you should load the latest pgi module, this is version 14.4.

$ module load pgi/pgi-14.4

This will provide you with the following compilers in your path

Language Compiler
C pgcc
Fortran pgfortran
C++ pgc++

GNU Compiler Collection

You need to load the GNU module, the latest version is 4.9.2

$ module load gcc/gcc-4.9.2

This will provide you with the following compilers in your path

Language Compiler
C gcc
Fortran gfortran
C++ g++
